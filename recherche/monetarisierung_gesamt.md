# Match3 Emoji Puzzle - Monetarisierungsstrategie

**Konsolidierter Bericht aus 3 Sales-Recherchen**
**Stand: 27. Maerz 2026**

---

## Executive Summary

Unser Match3-Emoji-Spiel positioniert sich als **ethische Alternative** zu Candy Crush & Co. Der Markt ist mit $12,8 Mrd. (2025) riesig, aber von wenigen grossen Playern dominiert. Unsere Differenzierung: **Kein Pay-to-Win, keine Lootboxen, Open Source, faire Mechaniken.**

**Empfohlenes Modell:** Gestufter Hybrid-Ansatz mit 3 Phasen.
**Realistischer Jahresumsatz:** 2.400 - 6.600 EUR (bei 10k Downloads/Monat).
**Kernaussage:** Der groesste Hebel ist Retention, nicht das Monetarisierungsmodell.

---

## 1. Marktlage

| Kennzahl | Wert |
|---|---|
| Globaler Match-3-Umsatz 2025 | **$12,8 Mrd.** |
| Prognose 2034 | **$28,6 Mrd.** (CAGR 9,3%) |
| Mobile-Anteil | 72,6% |
| Top-Spieler | Royal Match (~$125M/Mo), Candy Crush (~$100M/Mo) |

**Problem der Grossen:** Aggressive Monetarisierung, kuenstliche Schwierigkeitsspitzen, Energiesysteme, FOMO-Mechaniken, manipuliertes RNG.

**Unsere Chance:** "Das faire Match3 -- Spass ohne Abzocke."

### Zielgruppe

- 55-60% weiblich, Kernalter 25-54 Jahre
- 78% Casual Gamer (10-20 Min/Session)
- Durchschnittlich 18 Min taegliche Puzzle-Spielzeit
- Geringe Zahlungsbereitschaft bei Casual ($0-5/Monat)

## 3. Monetarisierungsmodelle im Vergleich

| Modell | ARPU (Lifetime) | Jaehrlich netto (10k DL/Mo) | Bewertung |
|---|---|---|---|
| Nur Spenden | 0,01-0,05 EUR | 84-420 EUR | Nicht nachhaltig |
| Cosmetic IAP | 0,15-0,60 EUR | 1.260-4.200 EUR | Fair, benoetigt Content |

| **Hybrid (Ads + IAP)** | **0,20-0,80 EUR** | **2.400-6.600 EUR** | **Empfohlen** |

## 4. Empfohlene 3-Phasen-Strategie

### Phase 1: Launch (Monat 1-3) -- Nur Rewarded Ads

| Element | Details |
|---|---|
| Preis | Kostenlos |
| Monetarisierung | Optionale Rewarded Video Ads |
| Ad-Anlaesse | Extra-Zuege nach Game Over, Hint, Tages-Bonus |
| Ziel | Downloads maximieren, Retention messen |
| Erwartung | 5-40 EUR/Monat |

### Phase 2: Growth (Monat 4-9) -- Hybrid (Ads + Cosmetic IAP)

| Element | Details |
|---|---|
| Neue Features | Cosmetic IAP-Shop + "Remove Ads" (3,99 EUR) |
| IAP-Items | 3-5 Emoji-Theme-Packs (je 0,99-1,99 EUR) |
| Saisonale Themes | Ostern, Sommer, Halloween, Weihnachten |
| Erwartung | 80-550 EUR/Monat |

### Phase 3: Mature (Monat 10+) -- Vollstaendiger Hybrid + Abo

| Element | Details |
|---|---|
| Neues Feature | Optionales VIP-Abo (1,99 EUR/Mo oder 9,99 EUR/Jahr) |
| Abo-Inhalt | Alle Themes, keine Ads, exklusive Levels |
| Supporter-Tiers | Kaffee (1,99), Supporter (4,99), Patron (19,99 EUR) |
| Erwartung | 400-2.000 EUR/Monat |

## 7. Vertriebsstrategie

### Plattform-Priorisierung

1. **P1:** Google Play + App Store (Mobile-Markt = 72,6%)
2. **P2:** Web-Demo (Flutter Web) + F-Droid (Open-Source-Community)
3. **P3:** Desktop (Windows/macOS/Linux), Snap Store, Flathub
4. **P4:** Steam (optional, Community-Features)

### Launch-Plan

1. **Soft Launch:** GitHub + F-Droid + Web-Demo (Monat 1-2)
2. **Mobile Launch:** Play Store + App Store + ASO (Monat 3-4)
3. **Skalierung:** Desktop + Marketing + Community (Monat 5+)

### Marketing (Zero-Budget)

- ASO (Keywords: "fair match3", "no ads puzzle", "open source game")
- Reddit: r/AndroidGaming, r/iosgaming, r/opensourcegames
- Hacker News / Heise: "Open-Source Match3 ohne Abzocke"
- Discord-Server fuer Community
- GitHub als Vertrauenssignal

## 8. Was wir NICHT tun

| Anti-Pattern | Warum nicht |
|---|---|
| Energie-System / Leben-Timer | Kuenstliche Frustration = Manipulation |
| Lootboxen / Gacha | Gluecksspiel-Mechanik, reguliert |
| Pay-to-Win Power-Ups | Zerstoert Fairness |
| Aggressive Interstitials | Vertreibt Spieler |
| FOMO-Angebote ("Nur noch 2h!") | Manipulativ |
| Consumable IAPs (kaufbare Coins) | Oeffnet Tuer zu Pay-to-Win |
